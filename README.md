# Testes de API com Postman - ServeRest

Este projeto contém testes de API desenvolvida no Postman, utilizando a API pública ServeRest para praticar requisições e testes em endpoints REST.

O objetivo do projeto é simular operações comuns de uma API, relacionadas com o usuário (como criar e logar usuários) e também gerenciamento de produtos (como listar, cadastrar, deletar, buscar por ID e editar produtos), utilizando verbos HTTP por meio da ferramenta Postman.

---

# Tecnologias utilizadas

* Postman
* API ServeRest

---

# API utilizada

A API utilizada neste projeto foi a ServeRest, uma API gratuita que simula uma loja virtual, criada para prática de testes de software.

Documentação oficial:
https://serverest.dev

---

# Estrutura da Collection

A collection foi organizada em pastas para facilitar a execução e visualização dos endpoints.

## Documentação

* **GET** Documentação da API

## Usuários

* **POST** Criar usuário

## Login

* **POST** Login

## Produtos

* **GET** Listar produtos
* **POST** Cadastrar produtos
* **GET** Buscar produto por ID
* **PUT** Editar produto por ID
* **DELETE** Deletar produto

---

# Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de praticar:

* Testes de API REST
* Utilização do Postman
* Organização de collections
* Execução de requisições HTTP
* Estrutura de respostas JSON
* Testes de endpoints

Foram explorados os principais métodos HTTP:

* **GET** → Buscar dados
* **POST** → Criar dados
* **PUT** → Atualizar dados
* **DELETE** → Remover dados

# Como executar o projeto

1. Clone este repositório:

```
git clone https://github.com/mouraoleticia23-lgtm/postman_serverest.git
```

2. Abra o **Postman**

3. Clique em **Import**

4. Selecione o arquivo da collection que está neste repositório

5. Execute as requisições individualmente ou utilizando o **Collection Runner**
